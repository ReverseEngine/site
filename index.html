Assembly
Nome Uso Descrição
mov mov destino, origem Copia um valor da origem para o destino
push push origem Adiciona a origem na pilha
add add destino, origem Soma destino + origem e armazena o resultado em destino
add add destino, origem Soma destino + origem e armazena o resultado em destino
call add label Push no endereço atual e jump para o label
lea lea destino, origem Load Effective Addres da origem para o destino
test test origem_1, origtem_2 Set condition codes according to origem_1 & origem_2
jne jne label Jump if not equal/nonzero
